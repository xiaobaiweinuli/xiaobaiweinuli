# 📝 博客文章展示模板样式

## 样式 1：简洁卡片式 🎴
```yaml
template: "$newline### 📖 [$title]($url)$newline> 📅 发布于 $publishedTime | 🔄 更新于 $updatedTime$newline"
```

**效果预览：**
### 📖 [使用DSU Sideloder实现安卓双系统](http://blog.xingshuang.xyz/...)
> 📅 发布于 2024年05月08日 04:14:01 | 🔄 更新于 2025年11月18日 16:13:28

### 📖 [PDF分割合并](http://blog.xingshuang.xyz/...)
> 📅 发布于 2024年04月26日 12:29:29 | 🔄 更新于 2025年11月18日 16:13:28

---

## 样式 5：分栏表格式 📊
```yaml
template: "$newline| 📝 [$title]($url) | 📅 $publishedTime | 🔄 $updatedTime |$newline"
```

**需要在注释标签前添加表头：**
```markdown
| 文章标题 | 发布时间 | 更新时间 |
|---------|---------|---------|
<!-- BLOG-POST-LIST:START -->
<!-- BLOG-POST-LIST:END -->
```

## 🎨 使用方法

在 `.github/workflows/blog-rss.yml` 中找到 `template:` 这一行，替换成你喜欢的样式即可！

**注意事项：**
- `$newline` 表示换行
- `$title` 是文章标题
- `$url` 是文章链接
- `$publishedTime` 和 `$updatedTime` 是我们自定义的时间变量
- `$position` 是文章序号（1, 2, 3...）
